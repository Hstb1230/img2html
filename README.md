# img2html
用PHP代码实现图片转HTML网页

说明
---
* PHP代码是针对上传文件的情况写的。用于其他情况时，请去掉前几行的判断代码，并修改```$path```的值
* 需要开启GD扩展，而且要设置```short_open_tag(支持 PHP 代码的短开始标签)```
* 建议```PHP ≥ 5.5```，否则会不支持某些格式图片(```webp```，```bmp```等)
* 处理```BMP```图片需要```PHP 7.2```以上版本，因制作时使用PHP 5.6，未添加相关代码
